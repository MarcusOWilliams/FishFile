Searching the database

There are two main ways to search for an entry in the database: The simplest way is to use the small search bar in the top right hand corner, this allows you to search for a specific fish tank by giving its stock number If you want a more detailed search you can use the search form on the home page, this will allow you to filter by a number of attributes of any entry and will return a list of all the entries that match your search
